# Cloud Canteen Website

Welcome to the premium Cloud Canteen website template. This design was specifically crafted to work with your "Wooden Food Table" background image.

### How to use your image:
1.  Locate the image you shared in the chat.
2.  Save it into this folder (`e:\Suna Docs\Projects\cloud-canteen`).
3.  Rename the image file to **`bg.jpg`**.

### Features:
- **Premium Glassmorphism**: Modern, translucent UI elements for a state-of-the-art look.
- **Responsive Design**: Works perfectly on mobile and desktop.
- **Scroll Animations**: Smooth fade-ins and interactive menu cards.
- **Lucide Icons**: High-quality vector icons for a professional feel.
- **Custom Typography**: Uses "Outfit" and "Playfair Display" for a gourmet aesthetic.

### Project Structure:
- `index.html`: Main structure.
- `style.css`: All the premium styling and animations.
- `script.js`: Interactive scroll and animation logic.
- `bg.jpg`: Your provided background image (user-provided).

### Viewing the Site:
Open `index.html` in your browser to see the result.
